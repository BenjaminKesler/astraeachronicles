Character perfection before plot structure. User's stated approach: fully develop each character until their psychology is rock solid, then let the characters drive plot choices rather than the other way around.

**Why:** Plot decisions made before characters are fully understood tend to create situations that require characters to behave inconsistently. Character-driven plot emerges more naturally and holds together better.

**How to apply:** When plot questions arise during character work, note them as open questions but don't force resolution. When a plot idea conflicts with a character's established psychology, the character wins. Don't push toward plot completeness until the character foundation is explicitly declared solid by the user.
