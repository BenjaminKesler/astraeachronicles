LOAD-BEARING FILTER (meta-principle)
  every profile question must trace forward to: plot | theme | emotional impact | structural function
  if you can't trace it → cut it
  e.g. favorite laundry detergent = filler; daily routine = load-bearing (gives author something to disrupt)

COMFORT → STRATEGIC DENIAL
  identify character's comfort mechanisms: routines, rituals, go-to cure for bad day, meaningful objects
  identification IS the setup — you're cataloguing what to withhold
  distinct from [[scene-earning]] planted resonance (that's seeding a payoff; this is mapping deprivation targets)
  author move: place comfort detail early → deny it at precisely the moments it would matter most
