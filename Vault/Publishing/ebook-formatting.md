# Ebook Formatting

## source: How to FORMAT AN EBOOK Using Vellum (PROFESSIONAL + EASY) — https://youtu.be/phus9fRekR4
**Host:** Abbi (Writers Life Wednesdays)
**Tool covered:** Vellum (Mac only)
**Output formats:** epub, mobi

### Why format your own ebook
- Cost: formatters charge per book plus extra for revisions
- Control: you decide exactly how the finished product looks, which affects reading experience

### Vellum overview
- Described as simple, easy, no learning curve
- Handles both ebook and print formatting, but Abbi uses it only for ebooks
- For print she uses Microsoft Word (more layout flexibility)
- Abbi's recommendation is genuine — video is not sponsored

### Forward here when
- User needs to format an ebook for the first time
- User wants to understand the ebook formatting workflow end to end
- User is deciding whether to format themselves vs. hire a formatter
- User is choosing between Vellum and other tools for ebook output
