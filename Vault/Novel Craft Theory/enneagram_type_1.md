# TYPE 1 — GUT · anger-managed

WOUND: fear of corruption/defect → code = dam for gut anger (not suppressed — managed). Inner critic judges self far harsher than others; outward judgment is overflow, not primary target.
MECH [the dam]: anger is pre-cognitive gut signal; principles come second as translation/justification. The rules tell the one where the anger is permitted to go — and more importantly, where it must be contained.
ARC: get their hands dirty. Break the rule → discover still good. Core confusion to resolve: imperfection ≠ corruption.
ENGINE: code vs. reality resistance. Key question = what does the code cost internally (not externally)?
TENSION: identity IS code → flexibility = slippery slope to becoming what they most fear → arc demands exactly what they're most terrified of
ULTIMATE ARC: mercy = completion of code, not exception to it — they don't abandon the code, they outgrow it
STRESS→4: code failed + world fell apart anyway → "what's the point of doing right?" — loses organizing principle entirely, not just sad
INT→7: genuine joy in imperfect world; stop white-knuckling

SHAPES
  redeem: break can be small/unglamorous | fatal (Ned Stark) | disillusionment without triumph — loss of clean code is not a clean loss
  tragedy: code wins. Javert = code has no category for genuine redemption → confronted with reality the framework cannot account for → logic not despair
  ultimate: mercy IS completion of code (hardest to write)

ANTAG: not lying to themselves — no hypocrisy crack to exploit. Argument strengthens them. Weakness: concrete outcome code couldn't have achieved (not argument).
  incorruptible enforcer (Javert) | righteous destroyer (Thanos) | true believer (Umbridge)

NOT
  → judges self FAR harsher; outward criticism is overflow — ask what they said to themselves about the same flaw before they said it to anyone else
  → CAN be funny: sardonic, world-weary, "biggest haters" — righteous anger applied to small things is often comic
  → gut anger first, reasoned argument second — the one who contains the temper and delivers a calm argument has *translated* the anger, not eliminated it
  → code ≠ always legalistic — can be duty/craft/relationship; what makes it 1-coded is rigidity and cost of breaking, not content

DX
  1. What's the code specifically — institution, tradition, or self-constructed?
  2. What chaos/injustice forged it?
  3. What does inner critic say when THEY fail their own standard (not what they say to others)?
  4. Gut anger beneath the principles — if dam broke, what?
  5. Where is code already costing something they don't fully acknowledge?
  6. The one compromise that would destabilize their identity — what forces it?
  7. What does "dirty hands" look like for this specific code? What are they afraid of becoming?
  8. When code produces wrong outcome — how do they explain without updating?
  9. What does the closest relationship demand that the code forbids?
  10. Four-stress: code failed, world fell apart anyway — what does collapse look like?
  11. Integration: what would genuine joy in the imperfect world look like for this character?
  12. Harder on others or themselves — where does the mismatch show on the page?
