# Universal Diagnostics
# principles that apply across the full arc — not beat-specific
# extracted from beat-specific files; apply at any beat, scene, or decision point
# cross-references: [[internal-conflict]], [[wound-protective-mechanisms]], [[plot-structure-outlining]]

SWAPPABILITY TEST
  if this moment would be equally devastating/meaningful for any protagonist → it's wrong
  applies at every level: hook design, scene decisions, Act 2 choices, disaster, climax
  always ask: what does this mean specifically to THIS character given their wound and misbelief?
  → [[plot-structure-outlining]] Act 3 for disaster-level application (most critical deployment)
  → [[Beats/01_hook]] for hook-level application

MIRROR NEURONS VS. IDENTIFICATION
  generic event → reader feels biochemical response from own psychology filling the gap
  character-specific event → reader inhabits the character's psychology
  distinction holds continuously — not only at the disaster; operates at every beat and scene
  "me vs. them" test:
    reader asking "how would I feel if that was me?" = mirror neurons only; inhabitation failed
    reader asking "how would I feel if I was them?" = genuine identification; wound model active
  failure mode label: SURVIVAL MODE — plot-only engagement; reader observes but doesn't inhabit
  → [[plot-structure-outlining]] Act 3 for disaster-level application
  → [[Beats/01_hook]] for hook-level application

FORESEEABILITY CYCLE (reader engagement engine)
  loop: impossible choice → partially visible answer → reader foresees what might happen → must find out if right
  surprise ENDS the cycle; foreseeability SUSTAINS it
  cliffhanger ≠ "you don't know what happens next" — "you can see what might happen and must find out"
  operates at scene level throughout the arc; apply at any beat
  → [[plot-structure-outlining]] Act 2B for original context

CHARACTER LENS AUDIT (planning tool)
  for any major decision point: is the character's misbelief actually shaping their choice?
  or are they executing the plot-correct move regardless of inner state?
  double-duty scene = plot does what sequence needs AND choice reveals inner state without telling
  if a decision looks dumb to the reader → characterization failure, not dramatic irony
  use after plot structure is roughed out; re-run at every major beat
  → [[plot-structure-outlining]] Act 2B for original context
